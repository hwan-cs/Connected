## 기능 🛠

- 회원가입 및 로그인 
- 프로필 설정 (프로필 사진, 배경화면, 상태 메세지 등)
- 친구 추가
- 채팅 
- 편리한 녹음 기능
- 배속 기능, 음성인식 텍스트 변환 기능
- 실시간 위치 공유
- 라이트 모드, 다크 모드 지원
- 계정 삭제
